# market-sentiment-trader-analysis

# Market Sentiment vs Trader Performance Analysis

## Objective
This project analyzes how Bitcoin market sentiment (Fear/Greed) impacts trader behavior and profitability.

---

## Dataset
1. Bitcoin Sentiment Data (Fear/Greed Classification)
2. Hyperliquid Trader Data

---

## Methodology

### Part A — Data Preparation
- Cleaned datasets
- Converted timestamps
- Merged on daily date
- Created performance metrics

### Part B — Analysis
- Compared PnL during Fear vs Greed
- Analyzed trade frequency
- Calculated win rate
- Generated visualizations

### Part C — Strategy Recommendations
- Designed sentiment-based trading strategy
- Suggested risk management improvements

---

## Key Insights
- Trader behavior changes significantly across sentiment regimes.
- Trading activity increases during Fear.
- Risk-adjusted performance varies between Fear and Greed.

---

## How to Run

Install dependencies:

pip install pandas matplotlib seaborn
